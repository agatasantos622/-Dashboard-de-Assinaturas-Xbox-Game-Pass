# 🎮 Dashboard de Assinaturas Xbox Game Pass

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da **DIO + TOTVS — Fundamentos de Engenharia de Dados e Machine Learning**.

O objetivo foi construir um dashboard executivo em Excel para análise de assinaturas do Xbox Game Pass, transformando dados brutos em informações estratégicas de negócio.

O dashboard permite visualizar indicadores importantes relacionados a:

* desempenho dos planos
* comportamento de renovação
* faturamento
* ticket médio
* rentabilidade das assinaturas

Todo o projeto foi desenvolvido utilizando recursos do Microsoft Excel, com foco em análise de dados, visualização executiva e storytelling.

---

# 🚀 Objetivos da Análise

O dashboard busca responder perguntas estratégicas como:

* Qual é o plano mais contratado?
* Qual plano gera maior faturamento?
* Clientes Ultimate gastam mais que clientes Core?
* Qual plano possui maior taxa de renovação?
* Qual tipo de assinatura gera maior rentabilidade?
* Como os clientes se distribuem entre os planos?
